--------------------------------------------------------------------------------------------
Q1. What is Apache Kafka? Why was it created?
--------------------------------------------------------------------------------------------
✅ Apache Kafka is a distributed event streaming platform used to:
	Publish and subscribe to streams of events
	Store events durably
	Process events in real time
	It works as a high-throughput, fault-tolerant, scalable message broker built on a distributed commit log architecture.

✅ Why Was Kafka Created?
	Kafka was originally developed at LinkedIn to solve:
		1️⃣ High Throughput Problem: Traditional systems couldn’t handle millions of events per second.
		2️⃣ Real-Time Data Processing: Batch systems were slow. They needed near real-time pipelines.
		3️⃣ Decoupling Microservices: Services needed asynchronous communication.
		4️⃣ Scalability: Needed horizontal scaling across machines.
		5️⃣ Durability & Fault Tolerance: Data should not be lost if a server crashes.
