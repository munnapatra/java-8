Q. How do you check if a table has a duplicate data?
Ans. 
1. select count(*) 'row_count', count(distinct product_id) from shipment_table
2. select product_id, count(*) from shipment_table group by product_id

Q. How to compare two tables?
Ans.
1. select t1.product_id, t1.spid, t2.product_id, t2.spid
case
  when t1.product_id is null then 'Data in Table 2 only'
  when t2.product_id is null then 'Data in Table 1 only'
  else 'Data in both tables'
end as [data status]
from firs_week_sale t1
full join second_week_sales t2 on t1.product_id=t2.product_id and t1.spid= t2.spid
